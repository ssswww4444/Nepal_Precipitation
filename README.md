# Nepal_Precipitation

Precipitation data of Nepal extracted from the following two sources:

NOAA
- Dataset: CHIRPS (https://data.noaa.gov/onestop/collections/details/0cedc8c4-89d9-481a-92c4-4cf491dd989b)
- Frequency: Daily
- Availability: 1981 - Present
- Format: binary data files (converted to CSV)
- Code: R

JAXA
- Dataset: GSMaP (https://sharaku.eorc.jaxa.jp/GSMaP/guide.html)
- Frequency: Hourly
- Availability: 2000-03-01 to Present
- Format: CSV
- Code: Javascript (Running with Google Earth Engine)
